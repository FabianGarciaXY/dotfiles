
# Nvim

#### Requirements:

```Bash
# Update packages
sudo apt update

# Add c compiler for some plugins
sudo apt install build-essential

# Install fzf for fzf-lua
sudo apt install fzf
```

### Lsp (Optional)
```bash
#  Install npm
sudo apt update
sudo apt install nodejs npm

# Add pyright
npm install -g pyright

# Validate
which pyright-langserver
```
